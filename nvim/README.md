# Neovim Configuration

This directory is the repository's LazyVim configuration. LazyVim supplies capability-aware LSP navigation/action mappings, format-on-save, and pairing; this configuration adds the CP workflow, its manual format mapping, language servers, and formatter choices. The installer points Neovim at it and bootstraps the required plugins and language tools. See the [usage guide](../docs/usage.md#neovim-workflow) for the configured workflow and mappings.
